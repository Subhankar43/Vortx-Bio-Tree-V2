# Vortx Portfolio 🌐

A modern **personal portfolio / social links page** for **Vortx (Subhankar)** built with **HTML, CSS, and JavaScript**.
It provides a clean interface where visitors can quickly access social profiles and projects.

---

## ✨ Features

* Minimal **portfolio landing page**
* **Animated avatar section**
* **Social media link cards**
* **Projects hub navigation**
* **Share button**
* **Toast notification for link copy**
* Modern **UI design with gradient background**
* Fully **responsive layout**

---
